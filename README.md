# Syncou

Plataforma inteligente de agendamentos e reuniões online sincronizada com o Google Agenda.

## Funcionalidades
- **Links Únicos de Agendamento**: Compartilhe o seu link amigável (`syncou.app/p/nome`) com seus clientes, ideal para colocar na bio do Instagram ou enviar via WhatsApp.
- **Sincronização com o Google Calendar**: Todos os agendamentos são sincronizados automaticamente na sua agenda em tempo real.
- **Gestão Completa de Disponibilidade**: Customize seus dias, horários e duração de sessões com facilidade.
- **E-mail & Password e Google Auth**: Opções flexíveis e seguras de login ajustadas às suas necessidades.

## Tecnologias Utilizadas
- React + TypeScript + Vite
- Tailwind CSS para estilização moderna e limpa
- Firebase para Firestore Database e Authentication
